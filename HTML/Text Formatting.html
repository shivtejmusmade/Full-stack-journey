🧠 HTML Text Formatting Tags

► Text formatting tags allow us to emphasize or style specific portions of text.
► Text formatting tags are used to visually style content in HTML.
► Some tags carry semantic meaning, while others are purely visual and non-semantic.

1. <b> - Bold Text (Visual Only)

Purpose :
➤ Highlights text by making it bold, but without giving it importance in meaning.
Syntax/Example:
<p>This is a <b>bold</b> word.</p>
Output :
This is a bold word.

2. <i> - Italic Text (Visual Only)

Purpose :
➤ Displays text in italics, used for stylistic purposes without emphasizing importance.
Syntax/Example:
<p>This is an <i>italic</i> word.</p>
Output :
This is an italic word.

3. <u> - Underlined Text

Purpose :
➤ Underlines text for visual styling, without implying emphasis or importance.
Syntax/Example:
<p>This is an <u>underlined</u> word.</p>
Output :
This is an underlined word.

4. <mark> - Highlighted Text

Purpose :
➤ Highlights text to bring attention to it, typically with a yellow background.
Syntax/Example:
<p>This is a <mark>highlighted</mark> word.</p>
Output :
This is a highlighted word.

5. <sub> - Subscript Text

Purpose :
➤ Displays text slightly below the normal line, commonly used in chemical formulas or math.
Syntax/Example:
<p>H<sub>2</sub>O is water.</p>
Output :
H2O is water.

6. <sup> - Superscript Text

Purpose :
➤ Displays text slightly above the normal line, often used for footnotes, exponents, etc.
Syntax/Example:
<p>E = mc<sup>2</sup></p>
Output :
E = mc2

7. <small> - Smaller Text

Purpose :
➤ Reduces the font size of the text, typically used for disclaimers or fine print.
Syntax/Example:
<p>All rights reserved <small>© 2025</small></p>
Output :
All rights reserved © 2025

8. <s> - Strikethrough Text (Visual Only)

Purpose :
➤ Visually strikes through text to show it's no longer relevant or valid (e.g., old prices).
Syntax/Example:
<p>Was <s>$50</s> now $30</p>
Output :
Was $50 now $30

9. <del> - Deleted Text

Purpose :
➤ Indicates content that has been removed or is no longer accurate.
➤ Visually rendered as strikethrough in most browsers.
Syntax/Example:
<p>This product was <del>$100</del></p>
Output :
This product was $100

10. <ins> - Inserted Text

Purpose :
➤ Represents newly added content.
➤ Typically underlined in browsers.
Syntax/Example:
<p>Now available for <ins>$80</ins></p>
Output :
Now available for $80

11. <big> - Enlarged Text (Obsolete)

Purpose :
➤ Makes the text slightly larger than surrounding text.
➤ Obsolete in HTML5 — use CSS instead.
Syntax/Example:
<p>This is <big>bigger</big> text.</p>
Output :
This is bigger text.

12. <tt> - Teletype Text (Obsolete)

Purpose :
➤ Renders text in a monospaced (typewriter-like) font.
➤ Deprecated — use <code> or CSS for monospaced styling.
Syntax/Example:
<p>This is <tt>teletype</tt> text.</p>
Output :
This is teletype text.
