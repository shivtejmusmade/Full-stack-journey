🌿 Frontend Technologies

HTML

HTML is the foundation of all web pages.
It structures content using elements and tags.

CSS

CSS styles your HTML content.
Customize colors, background, fonts, layouts, and make your website responsive.

JavaScript

JavaScript brings interactivity to websites. You can manipulate elements, handle events, and create dynamic content.

jQuery

jQuery is a fast, small, and feature-rich JavaScript library. It simplifies DOM manipulation, event handling, and AJAX calls.

Bootstrap

Bootstrap is a CSS framework for building responsive, mobile-first websites using pre-designed components.

React

React is a JavaScript library for building modern user interfaces with reusable components and state management.
